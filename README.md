# Bloodsugar

## POST /api/bloodsugar



	POST /api/bloodsugar


### Examples

Request

```
axios.post('/api/bloodsugar');
```

### Success Response

Response

```
    HTTP/1.1 201
{
       "message":"Thank you, bloodsugar data has been added" 
   }
```
### Error Response

Response

```
HTTP/1.1 400
{
    "message":"User does not exist"
}
```
